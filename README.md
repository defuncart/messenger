# messenger

A messaging application for mobile and desktop.

## Getting Started

To build the app locally, the following tools are required:

- Flutter stable >= 2.5
- Dart >= 2.14
- macOS:
    - Xcode >= 12.4
    - CocoaPods >= 1.11.0
- Web:
    - Chrome

## Code Generation

### Localizations

To add new localizations, update `assets_dev/loca/loca.csv` and run

```sh
sh bin/loca_generate.sh
```
